# Llamatik Code – Issue Tracker

This repository is used to report bugs, request features, and provide feedback for Llamatik Code, a local-first AI coding assistant for IntelliJ-based IDEs.

---

## 🚀 Before creating an issue

Please check the following:

- Search existing issues to avoid duplicates  
- Make sure you are using the latest plugin version  
- Try restarting the IDE (some issues are environment-related)

---

## 🐞 Reporting a bug

When reporting a bug, please include:

### 1. Description
- What happened?
- What did you expect to happen?

### 2. Steps to reproduce
Provide clear steps, for example:
1. Open project
2. Run agent on file X
3. Click “Apply changes”
4. Error appears

### 3. Environment
- IDE (e.g. IntelliJ IDEA 2026.1)
- OS (macOS / Windows / Linux)
- Plugin version
- Model used (e.g. Qwen3-14B, Gemma, custom)

### 4. Logs (very important)
If possible, include relevant logs:

- Help → Show Log in Finder/Explorer
- Look for errors related to Llamatik Code

---

## 💡 Feature requests

We welcome suggestions!

Please include:
- The problem you are trying to solve
- Why the current behavior is not enough
- A proposed solution (if you have one)

---

## ⚡ Tips for better issues

- Be specific and concise  
- Include screenshots or recordings if helpful  
- Mention if the issue happens consistently or intermittently  

---

## 🔒 Privacy note

Llamatik Code is local-first.  
Please do not include sensitive code unless you are comfortable sharing it.

---

## ❤️ Feedback

General feedback is welcome!  
If something feels confusing, slow, or missing — let us know.
